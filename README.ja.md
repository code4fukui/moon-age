# moon-age

月齢と月の満ち欠けを表示するWebコンポーネントです。

## デモ
https://code4fukui.github.io/moon-age/

## 機能
- 指定した日付の月齢を表示
- 月の満ち欠けを円グラフで視覚的に表現

## 使い方

```html
<script type="module" src="https://code4fukui.github.io/moon-age/moon-age.js"></script>
<moon-age></moon-age>
```

```html
<script type="module" src="https://code4fukui.github.io/moon-age/moon-age.js"></script>
<moon-age value="2030-01-01"></moon-age>
```

```html
<script type="module" src="https://code4fukui.github.io/moon-age/moon-age.js"></script>
<moon-age size="300"></moon-age>
```
